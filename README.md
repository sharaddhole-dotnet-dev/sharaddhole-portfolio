# Sharad Dhole - Portfolio Website

This is a personal portfolio website of Sharad Dhole built with React to showcase my experience, projects, skills, and certifications.

## Features

- Responsive design optimized for all devices
- Interactive UI with smooth scrolling
- Project showcase section
- Skills and experience timeline
- Contact form
- Deployed with GitHub Pages

## Technologies Used

- React.js
- CSS3
- JavaScript
- GitHub Actions for CI/CD
- GitHub Pages for hosting
